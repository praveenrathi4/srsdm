# Environment Configuration for SRSDM Website

## Required Environment Variables

Create a file named ".env" in the root directory with the following content:

EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
PORT=3000

## Email Setup Instructions

1. Use Gmail for email notifications
2. Enable 2-factor authentication on your Gmail account
3. Generate an "App Password":
   - Go to Google Account settings
   - Security > 2-Step Verification > App passwords
   - Generate a new app password for "Mail"
   - Use this password as EMAIL_PASS

## WhatsApp Setup (Optional)

For WhatsApp notifications, you'll need to:
1. Sign up for a WhatsApp Business API service (Twilio, etc.)
2. Add these variables to your .env file:
   WHATSAPP_API_KEY=your-api-key
   WHATSAPP_PHONE=+918233666311

## Testing

1. Run: npm install
2. Run: npm start
3. Open: http://localhost:3000
4. Test the contact form to verify email notifications

## Production Deployment

For production deployment:
1. Set NODE_ENV=production
2. Use a proper email service (SendGrid, Mailgun, etc.)
3. Configure actual WhatsApp Business API
4. Set up SSL certificate
5. Use a process manager like PM2
